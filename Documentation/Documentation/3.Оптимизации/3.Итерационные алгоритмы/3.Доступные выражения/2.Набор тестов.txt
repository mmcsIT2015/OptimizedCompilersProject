# Тесты, связанные с анализом доступных выражений

#### Тесты, демонстрирующие независимость определения множества kill от взаимной ориентации базовых блоков:

~~~~~~{.cs}
{
  int a, b, c, d;
  if (1)
  {
    int p = b + c;
  }

  b = a - d;
  int u = b + c;
}
~~~~~~

Здесь во множество kill для "верхнего" базового блока попадет выражение `b + c`.

#### Тесты, демонстрирующие зависимость расположения выражений внутри базового блока:

~~~~~~{.cs}
{
  int a, b, c, d;
  if (1)
  {
    int p = b + c;
  }

  int u = b + c;
  b = a - d;
}
~~~~~~

#### Заключение
Для тестирование написаны соответствующие методы, которые выводят ответы в доступной форме.

Оба алгоритма (генерация Gen-Kill и итерационный алгоритм) проверялись на примерах `test-exprgenkill-1.cn` ... `test-exprgenkill-5.cn`.
