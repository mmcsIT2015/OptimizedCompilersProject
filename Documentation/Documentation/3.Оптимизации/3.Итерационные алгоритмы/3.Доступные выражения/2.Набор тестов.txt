№ Тесты связанные с анализом доступных выражений

Присутствуют как тесты демонстрирующие независимость определения множества kill от взаимной ориентации
базовых блоков:

~~~~~~{.cs}
{
  int a, b, c, d;
  if (1)
  {
    int p = b + c;
  }

  b = a - d;
  int u = b + c;
}
~~~~~~

здесь во множество kill для "верхнего" базового блока попадет выражение b + c


Так и тесты демонстрирующие зависимость расположения выражений внутри базового блока:

~~~~~~{.cs}
{
  int a, b, c, d;
  if (1)
  {
    int p = b + c;
  }

  int u = b + c;
  b = a - d;
}
~~~~~~


Для тестирование написаны соответствующие методы, которые выводят ответы в доступной форме.

Оба алгоритма (генерация Gen-Kill и итерационный алгоритм) проверялись на примерах
