# Итерационный алгоритм для множеств Dom

Используется полурешетка с операцией пересечения. Элементами являются блоки.
Топовым элементом полурештки является множество всех блоков.

Передаточная функция представляет собой множество Gen с одним единственным элементом - данным блоком и пустое
множество Kill

~~~~~~{.cs}
var blockFuncGen = new HashSet<Block>() { block };
funcs[block] = new TransferFunction<Block>(blockFuncGen, emptyKill);
~~~~~~

На выходе получается набор множеств Dom для блоков. По смыслу, каждое множество
Dom - множество узлов, которые "доминируют" над данным. По доминирование подразумевается
то, что при обходе графа потока управления каждый элемент из множества Dom
будет присутстовать на любом из путей из начала графа в данный блок.
